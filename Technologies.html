Anaconda Navigator: Anaconda Navigator is a graphical user interface for managing packages, environments, and software associated with the Anaconda Python distribution, which is popular among data scientists.

Google Colab: Google Colab is a cloud-based platform that provides a free environment for running Jupyter notebooks with support for Python and machine learning libraries. It's commonly used for data analysis and machine learning.

TensorFlow: TensorFlow is an open-source machine learning framework developed by Google. It's widely used for building and training deep learning models.

Visual Studio: Visual Studio is an integrated development environment (IDE) from Microsoft. It supports various programming languages and is commonly used for developing desktop applications, web applications, and more.

Visual Studio Code:Visual Studio Code (VS Code) is a lightweight, open-source code editor developed by Microsoft. It's highly extensible and popular among developers for its versatility and wide range of extensions.

Sandbox: The term "sandbox" generally refers to a controlled environment used for testing and running untrusted or potentially harmful code safely without affecting the host system.In here we used Cuckoo Snadbox.

XAMPP: XAMPP is a free, open-source cross-platform web server solution stack that includes Apache, MySQL, PHP, and Perl. It's commonly used for local web development and testing.

MongoDB: MongoDB Atlas is a cloud-based, fully managed database service for MongoDB, a NoSQL database. It simplifies database deployment, scaling, and management.

Python: Python is a versatile, high-level programming language widely used in web development, data science, machine learning, and more.

HTML, CSS, JS: HTML (Hypertext Markup Language), CSS (Cascading Style Sheets), and JS (JavaScript) are fundamental technologies for web development. HTML is used for structuring web content, CSS for styling, and JS for adding interactivity.

Inno Setup: Inno Setup is a free, script-driven installation system created in Delphi for creating Windows installers. It's commonly used to package and distribute software for Windows.

PowerShell: PowerShell is a task automation and configuration management framework from Microsoft. It's used for scripting, automation, and managing Windows environments.
